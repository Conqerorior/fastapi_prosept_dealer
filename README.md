# Запуск проекта:
- Необходимо создать файл **.env**, в корневой папке проекта, с переменными окружения, для подключения к PostgreSQL
  ```
  DB_NAME=postgres
  POSTGRES_USER=postgres
  DB_HOST=localhost
  DB_PORT=5432
  POSTGRES_PASSWORD=admin1202
  ```
